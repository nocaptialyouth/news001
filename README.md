# EVER:READY — 현대오토에버 지원 원스톱 가이드

`index.html`을 더블클릭하면 바로 사용할 수 있는 정적 웹사이트입니다.

## 주요 기능

- 5개 직무 방향별 준비 증거·면접 질문·자기소개서 예시
- 브라우저에 자동 저장되는 지원 체크리스트, 자기소개서, STAR 메모
- 2026년 7월 13일 13:00 기준 마감 카운트다운
- 모바일 반응형 화면, 인쇄/PDF 저장, 웹 공유
- 현대오토에버 공식 채용·사업·FAQ·직무 인터뷰 링크

## 공유 방법

폴더 전체를 GitHub Pages, Netlify Drop, Cloudflare Pages 등 정적 호스팅에 업로드하세요. 파일을 직접 전달할 때는 폴더 전체를 ZIP으로 압축해야 이미지가 함께 보입니다.

## 주의

개인 지원 준비용 비공식 자료입니다. 모집 직무, 근무지, 자격, 전형 일정은 변동될 수 있으므로 제출 직전 공식 채용 공고를 확인하세요.
